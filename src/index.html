// O que é isso?

TesteLab é um frontend consumindo uma API REST de conceitos sobre testes de software. Cada conceito traz uma descrição completa, palavras-chave, exemplos práticos e relacionamentos com outros conceitos. Navegue por todas as entradas, pesquise por termos ou explore um conceito aleatório para expandir seu conhecimento.

// API Backend

Servidor Express.js com persistência em data.json. Todos os endpoints seguem REST:

GET/items— lista todos os conceitos
GET/items/:id— busca por ID
POST/items— cria novo conceito
PUT/items/:id— substitui conceito completo
PATCH/items/:id— atualização parcial
DELETE/items/:id— remove conceito

// Categorias

Os conceitos estão organizados em quatro categorias: Tipos de Testes (o quê testar), Níveis de Testes (onde testar), Técnicas de Testes (como testar) e Práticas de Testes (como se organizar para testar). Use a página Geral para filtrar por categoria.
